Validation & Persistence
System validates all details, stores student & parent data, uploads documents securely. Registration link invalidated → no further changes allowed.
If validation error → show error, request correction (parent edits & resubmits)
On success → registration saved, link expired
System
send confirmations

⏱️ 48-Hour Expiry Rule
Unique registration link expires after 48h. Parent must complete before deadline.
🔀 Decision Points
Link validity check (valid/expired) + validation success/error correction
✅ Post-registration
Link invalidated, no edits. BM notified, student receives SMS.
